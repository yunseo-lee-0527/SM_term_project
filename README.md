# StudyFlow Notes

Goodnotes와 유사한 디지털 필기 앱 흐름에 과학적 관리 개선 기능을 더한 Term Project 프로토타입입니다.

## 주요 기능

- 라이브러리, 문서 탭, 페이지 썸네일, 상단 필기 툴바가 있는 앱형 UI
- 펜, 형광펜, 지우개, 올가미, 텍스트 입력
- 펜/지우개 빠른 전환
- 글씨 두께와 지우개 크기의 연속 조절
- 강의, 문제풀이, 실험, 빈 종이 템플릿
- 이과생용 도형 요소: 좌표축, 정규분포, 사인파, 포물선, 벡터, 행렬, 분자 구조, 표
- 삽입한 요소를 길게 눌러 다시 편집하는 객체형 Elements
- 벡터 시작점/끝점 핸들 조정, 표 행/열/색상 조정, 행렬 자리 가이드, 좌표축 1D/2D/3D 선택
- 이미지 업로드 및 자료 보관함 삽입
- Ctrl+휠 또는 두 손가락 핀치 기반 확대
- 필기 중 휴식 타이머
- 도구 전환, 색 변경, 두께 변경, 도형 삽입, 올가미, 확대 제스처, 휴식 횟수 기록

## 실행

브라우저에서 `index.html`을 열면 바로 실행됩니다.

## 배포

이 저장소는 정적 웹앱으로 구성되어 있으며, GitHub Pages 배포용 워크플로우가 포함되어 있습니다.

예상 배포 주소:

`https://yunseo-lee-0527.github.io/SM_term_project/`

GitHub 저장소의 Pages 설정에서 Source를 `GitHub Actions`로 두면 `main` 브랜치에 push될 때 자동 배포됩니다.

## 프로젝트 의도

단순 필기 앱 비교가 아니라, 디지털 강의 필기를 인간-도구 작업 공정으로 보고 VA/NVA/Therblig/ECRS 관점에서 개선안을 제안하기 위한 프로토타입입니다.
